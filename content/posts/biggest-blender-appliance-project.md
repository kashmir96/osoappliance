They created an automated blender robot called “AutoBlend” that was capable of creating customized smoothies based on user input and preferences. The robot could scan ingredients placed into its container and determine which recipe would best fit those ingredients using artificial intelligence algorithms. The AutoBlend system also included an accompanying app that allowed users to customize their order before sending it off for blending.
